## 项目介绍
这是一个基于自定义武器评分的斯普拉遁3鲑鱼跑模式的分析网页

### 使用方法
- 通过在武器页面的各个武器进行自定义打分，所有打工场次都会基于自定义的武器评分进行计算
- 自定义的打分结果文件可以下载下来，分享给需要的朋鱿

### 数据来源
- 往期打工数据基于[Splatoon3 Wiki (JP)](https://wikiwiki.jp/splatoon3mix/)
- 最新数据基于[Splatoon3.ink](https://splatoon3.ink/)

## Project Introduction
This is a Splatoon 3 Salmon Run analysis website based on customized weapon ratings.

### How to Use
- By assigning custom scores to various weapons on the weapon page, all Salmon Run shifts will be calculated based on these personalized ratings.
- The custom scoring file can be downloaded and shared with other inklings and octolings who might need it.

### Data Sources
- Historical Salmon Run data is based on [Splatoon3 Wiki (JP)](https://wikiwiki.jp/splatoon3mix/).
- The latest data is based on [Splatoon3.ink](https://splatoon3.ink/).

## プロジェクト紹介
これはカスタム武器評価に基づいた『スプラトゥーン3』サーモンラン専用の解析ウェブサイトです。

### 使用方法
- 武器ページにて各武器にカスタム評価（スコア）を付けることで、すべてのバイトスケジュールがその評価に基づいて計算されます。
- カスタム評価の結果ファイルはダウンロードして、他のバイターの仲間たちに共有することが可能です。

### データソース
- 過去のシフトデータは [Splatoon3 Wiki (JP)](https://wikiwiki.jp/splatoon3mix/) に基づいています。
- 最新データは [Splatoon3.ink](https://splatoon3.ink/) に基づいています。
